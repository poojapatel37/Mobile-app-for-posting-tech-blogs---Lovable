# Mobile-app-for-posting-tech-blogs---Lovable
A mobile application that allows users to create, publish, and browse tech blogs, designed with a clean interface and smooth content posting experience.

<img width="975" height="445" alt="image" src="https://github.com/user-attachments/assets/b84e6ff6-4c66-4390-a7fa-b1fc00e99908" />

## Main Functions
- User can create an account
- Admin can post blogs, delete them, delete comments from other users
- User can read blogs, add comments, like blogs, delete or edit their comments
